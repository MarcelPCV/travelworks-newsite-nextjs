# TravelWorks Typography Tokens

_Reference for the global Poppins typography system and utility classes._

## 1. Font Family

- **Primary Sans**: `Poppins`
- **Token**: `--font-poppins`
- **Tailwind Usage**: `font-sans`

## 2. Type Scale Tokens

| Token Class | Target | Size | Line Height | Weight |
| --- | --- | --- | --- | --- |
| `.type-h1` | Heading 1 | `clamp(2.5rem, 5vw, 3.75rem)` | `1.1` | `700` |
| `.type-h2` | Heading 2 | `clamp(2rem, 4vw, 3rem)` | `1.12` | `700` |
| `.type-h3` | Heading 3 | `clamp(1.75rem, 3.2vw, 2.5rem)` | `1.15` | `700` |
| `.type-h4` | Heading 4 | `clamp(1.5rem, 2.7vw, 2rem)` | `1.2` | `700` |
| `.type-h5` | Heading 5 | `clamp(1.25rem, 2.2vw, 1.5rem)` | `1.25` | `700` |
| `.type-h6` | Heading 6 | `clamp(1.125rem, 2vw, 1.25rem)` | `1.3` | `700` |
| `.type-normal-16` | Body text | `1rem` (16px) | `1.6` | `400` |

## 3. Font Weight Tokens

| Weight Name | Numeric |
| --- | --- |
| Thin | `100` |
| ExtraLight | `200` |
| Light | `300` |
| Regular | `400` |
| Medium | `500` |
| SemiBold | `600` |
| Bold | `700` |
| ExtraBold | `800` |
| Black | `900` |

## 4. Usage Snippets

```tsx
<h1 className="type-h1">Page Title</h1>
<p className="type-normal-16">Readable body text.</p>
```

```tsx
<section className="font-sans">
  <h2 className="type-h2">Feature Heading</h2>
  <p className="type-normal-16">Supports all Poppins weights and italic style.</p>
</section>
```
